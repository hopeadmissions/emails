Congrats on being accepted to Hope College!

As I’m sure you already know, one important part of a Hope education is foreign language study, and each student needs to complete the foreign language requirement. You’ve already had some German, and from your high school transcript, the best place for you to start would probably be German 102 (offered each Spring).

We use a textbook for German 101 and 102 that I helped develop called Auf geht’s! which focuses a great deal on culture and not so much on grammar. You learn a lot about how Germans relate, how they see friendships, the many difference between the various German regions, etc.

If you feel like you need a refresher, it's usually OK to start in German 101, which is offered each fall. It usually moves quite a bit faster than high school German, so it may be a good choice for you. If you start in 102 and pass it (of course!) you will also get 4 free credits for German 101 (as if you had taken it).

Please feel free to email me if you have any questions/comments – and let me know if you are planning on taking 102 in the spring. I hope to see you sometime next year!
